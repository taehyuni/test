Trouble & Optimization · 시행착오
개발 중 마주친 문제와 성능 최적화
해결됨 디스크 공간 부족 (errno 112)
Windows C 드라이브 용량 부족으로 flutter pub get 실패. Temp 폴더 위치 문제.
→ PUB_CACHE 환경변수를 D 드라이브로 이전. 향후 같은 문제 영구 차단.
해결됨 spread 문법 인코딩 문제
Dart의 if(...) ...[ spread 문법이 한글 환경 압축에서 깨지는 현상.
→ 명령형 List<Widget>.add() 방식으로 리팩토링. 인코딩 비의존.
최적화 API 응답 시간 단축
초기에 항공 → 숙소 순차 호출 시 평균 3.2초 소요.
→ Future.wait() 병렬 호출로 1.5초로 단축. 53% 개선.
최적화 렌더링 성능
탭 전환 시마다 위젯 재생성으로 끊김 발생.
→ IndexedStack + ConsumerWidget으로 상태 보존. 60fps 유지.
